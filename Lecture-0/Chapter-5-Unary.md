## 📝 Handnotes

**Unary** is a **base-one number system**, where each number is represented by repeatedly using a single symbol.

For example, the number **3** can be expressed as `|||`.

The familiar **tally mark system** is essentially a form of this unary system. However, in **pure (theoretical) unary**, the number 5 is written as `|||||` — without any diagonal crossbar.

> **💡 Note:** The diagonal crossbar (crossing every 5th mark) is a human convenience added to the tally system to make counting easier. In strict unary, no such shortcut exists.

---

## 🔑 Key Takeaways

- ✅ Unary is a **base-one** number system
- ✅ Each number = a **symbol repeated** that many times
- ✅ The **tally mark system** is a practical variant of unary
- ✅ Pure unary uses **no diagonal crossbar** — just straight marks
- ✅ The crossbar in tally marks is a **human convenience**, not part of true unary
